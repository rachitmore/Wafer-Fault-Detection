Computer Vision Project

Micro-Expression Recognition with Deep Learning

Objective:

Building a machine learning model to analyze micro-expressions in human faces and accurately predict subtle facial expressions.

Role:

Sole Developer, responsible for data collection, preprocessing, model development, and evaluation.

Technologies Used:

Python, TensorFlow, Pandas, Matplotlib, numpy, etc.


Current Progress and Key Learnings:

  • Successfully collected and curated a dataset of micro-expressions from publicly available sources, ensuring diversity and representativeness.
  • Completed data preprocessing, including face detection, normalization, and augmentation, to enhance the model's ability to generalize across varied facial expressions.
  • Developed and fine-tuned convolutional neural networks (CNNs) using TensorFlow, achieving promising preliminary results in detecting subtle micro-expressions.
  • Gained hands-on experience in using advanced deep learning techniques, such as transfer learning, to improve model accuracy with limited data.

Future Plans:

  • Plan to further optimize the model's performance by experimenting with different network architectures and hyperparameters.
  • Implement real-time micro-expression recognition using video data and deploy the model in a web-based application for practical use.
  • Explore the integration of the model with emotion recognition systems to provide more comprehensive insights into human behavior.
